# 第13章:社区管理功能实现

> **本章导读**
>
> 社区(Community)是论坛的基础组织单元。用户在不同社区下发布帖子、参与讨论。
> 本章将实现两个社区相关接口:**获取社区列表**和**获取社区详情**。

---

## 1. 数据库模型设计

### 1.1 internal/model/community.go

我们使用 GORM 的 `gorm.Model` 来集成基础字段（ID, CreatedAt, UpdatedAt, DeletedAt）。

```go
package model

import "gorm.io/gorm"

// Community 社区结构体
type Community struct {
	gorm.Model
	CommunityName string `gorm:"column:community_name;not null;size:255"`
	Introduction  string `gorm:"column:introduction;not null;type:text"`
}

func (Community) TableName() string {
	return "community"
}
```

---

## 2. DTO (数据传输对象)

为了避免直接暴露数据库模型, 并解决 JavaScript 的 `int64` 精度问题，我们使用 DTO。

#### 文件: `internal/dto/response/community.go`

```go
package response

import "time"

// CommunityResponse 返回给客户端的社区信息
type CommunityResponse struct {
	ID           string    `json:"id"` // 转换为字符串，防止 JS 精度丢失
	Name         string    `json:"name"`
	Introduction string    `json:"introduction,omitempty"`
	CreateTime   time.Time `json:"create_time"`
}
```

---

## 3. Service 层实现

Service 层负责将 Model 转换为 DTO。

#### 文件: `internal/service/community/community_service.go`

```go
// toResponse 将 model.Community 转换为 response.CommunityResponse
func toResponse(c *model.Community) *response.CommunityResponse {
    return &response.CommunityResponse{
        ID:           strconv.FormatUint(uint64(c.ID), 10), // 使用 gorm.Model 的自增 ID
        Name:         c.CommunityName,
        Introduction: c.Introduction,
        CreateTime:   c.CreatedAt,
    }
}

// GetCommunityList 获取社区列表
func (s *communityServiceStruct) GetCommunityList(ctx context.Context) ([]*response.CommunityResponse, error) {
    data, err := s.communityRepo.GetCommunityList(ctx)
    if err != nil {
        // DAO 层返回的错误已被 errorx.Wrap 包装，
        // 这里直接返回 err，由上层统一处理
        return nil, err
    }

    result := make([]*response.CommunityResponse, 0, len(data))
    for _, c := range data {
        result = append(result, toResponse(c))
    }
    return result, nil
}
```

> **错误处理说明**: Service 层直接返回 DAO 层返回的错误。DAO 层使用 `errorx.Wrap()` 包装数据库错误，保留了错误链和业务错误码。
