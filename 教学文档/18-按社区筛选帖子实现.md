# 第18章:按社区筛选帖子实现

> **本章导读**
>
> 现在我们已经实现了全站帖子的投票和排序,但用户还需要一个重要功能:**按社区筛选帖子**。

## 1. 参数设计

我们通过 `request.PostListRequest` 统一获取参数。

type PostListRequest struct {

}

## 2. Logic 层实现 - 调度器模式

Service 层会根据 `CommunityID` 是否为 0 来决定查询逻辑。

#### 文件: `internal/service/post/post_service.go`

```go
func (s *PostService) GetPostList(ctx context.Context, p *request.PostListRequest) (data []*response.PostDetailResponse, err error) {
	var posts []*model.Post
	if p.CommunityID == 0 {
		// 全站查询
		posts, err = s.postRepo.GetPostList(ctx, p.Page, p.Size)
	} else {
		// 按社区查询
		posts, err = s.postRepo.GetPostListByCommunityID(ctx, p.CommunityID, p.Page, p.Size)
	}
	// ... 后续组装逻辑 ...
}
```

## 3. DAO 层实现

#### 文件: `internal/dao/mysql/post.go`

```go
func (r *PostRepo) GetPostListByCommunityID(ctx context.Context, communityID, page, size int64) (posts []*model.Post, err error) {
	err = r.db.WithContext(ctx).
		Preload("Author").
		Preload("Community").
		Where("community_id = ?", communityID).
		Order("created_at DESC").
		Limit(int(size)).
		Offset(int((page - 1) * size)).
		Find(&posts).Error
	return
}
```
